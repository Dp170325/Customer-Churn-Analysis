# Customer Churn Analysis using SQL Server & Power BI

## Project Overview

This project analyzes customer churn behavior in a telecom dataset using SQL Server and Power BI. The objective is to identify factors contributing to customer attrition, evaluate revenue impact, and provide actionable business insights through interactive visualizations.

---

## Objectives

* Analyze customer churn trends and patterns.
* Identify major reasons for customer attrition.
* Measure the financial impact of churn.
* Segment customers based on demographics and service usage.
* Develop an interactive Power BI dashboard for decision-making.

---

## Dataset Information

* Total Customers: 6,418
* Domain: Telecommunications
* Data Source: Customer Churn Dataset

---

## Technologies Used

* SQL Server
* SQL Server Management Studio (SSMS)
* Power BI

---

## SQL Analysis Performed

### Data Cleaning

* Handled missing values using `ISNULL()`
* Created staging and production tables
* Performed data quality checks

### Exploratory Data Analysis (EDA)

* Customer Status Analysis
* Revenue Analysis
* Contract Analysis
* Internet Service Analysis
* Age Group Analysis
* Churn Category Analysis
* Churn Reason Analysis

### Advanced SQL Concepts

* Common Table Expressions (CTEs)
* Views

---

## Key Insights

* Total Customers: 6,418
* Churned Customers: 1,732
* Customer Churn Rate: 26.99%
* Revenue Lost Due to Churn: 3.41 Million
* Competitor-related issues accounted for the largest share of customer churn.
* Retained customers contributed the majority of total revenue.
